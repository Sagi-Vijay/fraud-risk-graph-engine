# Fraud Risk Graph Engine

A zero-cost, cloud-based fraud detection project focused on transaction fraud, identity fraud, graph analytics, risk scoring, and business decisioning.

## Current status
This repository is being built phase-by-phase for a realistic, interview-ready fraud analytics portfolio project.

## Stack
- Python
- pandas, numpy, scikit-learn, xgboost, networkx
- Streamlit
- GitHub
- Google Colab
- Optional Azure Functions / Static Web Apps

## Planned capabilities
- Synthetic fraud data generation
- Behavioral and velocity features
- Graph-based fraud detection
- Tabular + graph model comparison
- Risk score + approve/review/decline decisioning
- Streamlit case-review app

## Run locally
```bash
pip install -r requirements.txt
streamlit run src/app/streamlit_app.py
```
